# Lotto Number Generator Project Blueprint

## Overview
This project is a modern, responsive web application designed to recommend Lotto numbers (6/45 format). It operates within a Firebase Studio environment using standard HTML, CSS, and JavaScript. The goal is to provide a visually engaging user experience with animations and clear data visualization.

## Current Version Features
- **Number Generation:** Generates 6 unique random numbers between 1 and 45.
- **Visual Display:** Renders numbers as colored balls, mimicking real Lotto balls with entry animations.
- **Color Coding:**
  - 1-10: Yellow (Standard Lotto Color)
  - 11-20: Blue
  - 21-30: Red
  - 31-40: Gray
  - 41-45: Green
- **History:** Displays a list of the last 5 generated sets.
- **Responsive Design:** Adapts layout and ball size for mobile and desktop screens.
- **Modern UI:** Uses CSS variables, gradients, and shadow effects for a polished look.

## Technical Details
- **Entry Point:** `index.html`
- **Styling:** `style.css` (Vanilla CSS with variables and animations)
- **Logic:** `main.js` (ES Modules, DOM manipulation)
