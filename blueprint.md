# Lotto Number Generator Project Blueprint

## Overview
This project is a modern, responsive web application designed to recommend Lotto numbers (6/45 format). It operates within a Firebase Studio environment using standard HTML, CSS, and JavaScript.

## Current Version Features
- **Number Generation:** Generates 6 unique random numbers between 1 and 45.
- **Visual Display:** Renders numbers as colored balls with entry animations.
- **Color Coding:** Standard Korean Lotto colors (1-10 Yellow, 11-20 Blue, 21-30 Red, 31-40 Gray, 41-45 Green).
- **History:** Displays the last 5 generated sets.
- **Theme Support:** **Dark Mode and Light Mode** toggle with persistent storage.
- **Responsive Design:** Mobile and desktop optimized.

## Technical Details
- **Entry Point:** `index.html`
- **Styling:** `style.css` (CSS Variables for theme management)
- **Logic:** `main.js` (Includes theme switching and randomization)
